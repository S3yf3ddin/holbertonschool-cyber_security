# 0x02 Forensic Methodologies

This project introduces **digital forensics** — the discipline of identifying, preserving, analysing, and presenting digital evidence. Students will learn how to investigate a simulated security incident by examining artefacts, determining the attacker's intent, identifying the vulnerability exploited, and understanding the wider impact of the breach. Forensic methodologies are a critical part of any incident-response programme.

## 🎯 Learning Objectives

At the end of this project, you are expected to be able to explain the following to anyone, **without the help of Google**:

* What are the **phases of a digital forensics investigation**?
* How do you maintain a proper **chain of custody**?
* How can **file metadata** reveal information about a security incident?
* What is **steganography** and how is it used to hide data?
* How do you determine an **attacker's intent** from forensic artefacts?
* What types of **vulnerabilities** are commonly exploited in attacks?
* How does one incident create a **ripple effect** across an organisation?

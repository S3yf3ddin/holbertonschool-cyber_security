# Network Security

This directory contains a series of projects on **network security** — the practice of protecting computer networks from unauthorized access, misuse, and attacks. The modules progress from passive information gathering and active reconnaissance through packet analysis, host discovery, and advanced port scanning techniques. These skills are fundamental to penetration testers, network defenders, and security analysts.

## 📂 Modules

| Directory | Topic |
|-----------|-------|
| `0x01_passive_reconnaissance/` | Gathering information without direct target interaction (WHOIS, DNS, subdomain enumeration) |
| `0x02_active_reconnaissance/` | Direct target interaction to map open ports, web servers, and injection points |
| `0x04_nmap_live_hosts_discovery/` | Using Nmap ping techniques to discover live hosts on a network |
| `0x05_wireshark_basics/` | Capturing and analysing packets with Wireshark display filters |
| `0x06_nmap_advanced_port_scans/` | Stealth and advanced scan types: NULL, FIN, XMAS, Maimon, ACK, Window |
